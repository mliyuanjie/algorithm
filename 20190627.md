# 排序 
## quicksort  
**选择第一个点i，往后找小于它的点t，替换，i=i+1，t++** 
```
void Sort::quicksort(int s, int e){
	if((e-s)<=30){insertsort(s,e);return;}
	int tmp=array[s];
	int t=s;
	for(int i=s+1;i<=e;i++){
		if(array[i]<tmp){
			array[t]=array[i];
			array[i]=array[t+1];
			t++;
		}
	}
	array[t]=tmp;
	quicksort(s,t-1);
	quicksort(t+1,e);
	return;
}
```
